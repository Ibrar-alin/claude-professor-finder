# Example: Machine Learning Research

## User Query
Find 3 emails of professors at Stanford in the USA working in the field of deep learning for computer vision.


## Expected Behavior

The skill should:
1. Search Stanford University faculty profiles
2. Identify professors in Computer Science, EECS, or related departments
3. Focus on those with research in deep learning and computer vision
4. Extract publicly available emails
5. Find recent papers matching the research area
6. Return no more than 3 professors
